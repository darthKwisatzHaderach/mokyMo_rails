# MokyMo: test manager

MokyMo is a simple test manager. MokyMo allows you to create test suites and test cases.

## Install

```shell
git clone https://github.com/darthKwisatzHaderach/mokyMo
cd mokyMo
bundle install
rake db:create
rake db:setup
rails s
```

## Use

Open browser. Go to http://localhost:3000.
